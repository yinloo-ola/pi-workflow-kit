# Contributing

Thanks for taking the time to contribute!

## Where to post what (strict)

- **Questions / support:** use **GitHub Discussions**
  - https://github.com/coctostan/pi-superpowers-plus/discussions
- **Bugs:** open a **GitHub Issue**
- **Feature requests:** open a **GitHub Issue**
  - https://github.com/coctostan/pi-superpowers-plus/issues/new/choose

If a question/support request is opened as an Issue, it may be **closed** and redirected to Discussions.

## Reporting bugs

Please include:
- clear **steps to reproduce** (copy/paste commands if possible)
- **expected vs actual** behavior
- your environment:
  - OS
  - Node version
  - `pi` version
- relevant logs/output (please redact secrets)

## Requesting features

Please focus on:
- the problem you’re trying to solve (user story is great)
- what “done” looks like
- constraints / alternatives you’ve considered

## Development setup

Requirements: recent Node.js (LTS recommended)

```bash
npm install
npm test
```

## Pull request guidelines

- Keep PRs small and focused.
- Add or update tests when behavior changes.
- Update docs when user-facing behavior changes.
- Link related Issues/Discussions in the PR description.
- If you’re unsure about approach/scope, start with a Discussion first.
